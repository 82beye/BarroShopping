# 바로쇼핑 쇼츠 스크립트 생성 (P2-4 / FR-2)

상품 데이터를 받아 쇼핑 쇼츠(세로 9:16) 스크립트를 생성한다. Anthropic API가 아니라 이 Claude Code 스킬로 작성한다.

## 입력
- 상품 데이터 JSON: `name`(1~2줄 배열 또는 문자열) · `category` · `sub`(스펙) · `was`/`now`(정가/판매가) · `rating` · `reviews` · `image` 등 (render `productSchema` 호환)
- 스타일: `정보형` | `감성` | `다이나믹`

## 출력 — JSON만 (설명·코드펜스 없이)
```
{
  "hook": ["첫 줄", "둘째 줄"],
  "scenes": [{"narration": "씬 나레이션", "caption": "화면 자막"}],
  "cta": "행동 유도 문구"
}
```

## 규칙
- 길이: 씬 narration 합산 15~30초 분량.
- **글자 길이 제한(영상 디자인 영역 초과 방지 — 길면 영상에서 글자가 작아짐):**
  - `hook` 각 줄 **≤ 12자**, `cta` **≤ 18자**, `caption` **≤ 16자**.
- 톤: 정보형=스펙·혜택 중심 / 감성=라이프스타일·공감 / 다이나믹=속도·임팩트.
- **가격·할인율은 입력값만 사용**. 과장·허위·미검증 효능 주장 금지 (PRD §14 저작권·법적 위험).
- 한국어. 출력은 위 JSON 한 덩어리만.

## 사용
- 수동: Claude Code에서 상품 JSON과 함께 이 스킬 호출.
- 자동: `packages/workers/script.py` 가 `claude -p` 헤드리스로 호출(워커 파이프라인 FR-2 단계).
