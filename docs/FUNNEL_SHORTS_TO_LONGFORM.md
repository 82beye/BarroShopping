# 쇼츠 → 롱폼 퍼널 가이드 (링크 우회 전략)

> YouTube가 **쇼츠 댓글·설명의 링크를 비클릭 처리**(2025-08-31~)하면서 프로필 링크만으로는
> 전환 유도가 약하다. 해결책: **쇼츠를 롱폼에 연결**하고, 클릭 가능한 링크는 **롱폼 쪽**에 둔다.

## 1. 왜 이 구조인가 (사실관계)

- **쇼츠 댓글/설명 링크 = 비클릭(텍스트)** — 어필리에이트 유도가 막힘.
- **쇼츠 → "관련 동영상"(Related video) 1개 연결 가능** — Studio에서 지정, 채널명 아래 클릭 링크 노출. *고급 기능 액세스* 필요.
- **롱폼은 다르다:**
  - **설명란 링크가 클릭됨** (YPP 불필요) ← 가장 확실한 우회로
  - **카드(Cards)** · **끝화면(End screen)** 부착 가능 (외부 사이트 링크는 YPP + "연결된 사이트" 인증 필요)
- **⚠️ 결정적 제약:** 목적지 영상이 **세로 3분 이하면 YouTube가 다시 쇼츠로 분류** → 똑같이 링크 불가.
  그래서 목적지는 **가로 16:9** 또는 **세로 3분 초과**여야 한다. (이 프로젝트의 `ProductLong` = 16:9)

```
쇼츠(후킹·9:16) ──[관련 동영상]──▶ 롱폼(16:9, ProductLong)
                                      ├─ 설명란 클릭 링크 ──▶ 랜딩(barroshopping.pages.dev) ──▶ 제휴몰
                                      ├─ 카드(중간)        ──▶ 랜딩/상품
                                      └─ 끝화면(마지막 20초)──▶ 구독 · 관련영상 · (YPP면)링크
```

## 2. 롱폼 만들기 (ProductLong, 16:9, 60~90초)

```bash
cd packages/workers

# (권장) 이미 만든 쇼츠 스토리보드를 그대로 재사용 — 비전 재분석 비용 0, 쇼츠와 스토리 일관
python samples/build_long.py --id <상품id> \
  --from-reel ../render/out/Product/<상품id>/reel.props.json \
  --affiliate <구매링크> --render

# 또는 원본 이미지로 새로 분석
python samples/build_long.py --id <상품id> --images a.jpg b.jpg c.jpg \
  --title "상품명" --affiliate <구매링크> --target-seconds 75 --render
```

산출물(`out/Product/<id>/`): `long.mp4` · `long.cover.png` · `long.props.json` · `long.publish.md`.
**기존 쇼츠 자산(video.mp4·publish.md)은 건드리지 않는다.**

- 구성: 인트로(히어로+훅) → 본문 컷(좌 이미지·우 자막 분할) → **끝화면 세이프존(마지막 ~20초, 본문을 가리지 않게 비워둠)**.
- 끝화면 요소 배치 위치를 영상에 점선으로 미리 보고 싶으면 `long.props.json`에 `"endcardGuides": true` 추가 후 재렌더(기획용, 최종본은 false).
- 커버는 자동 생성되지만 **커스텀 썸네일 업로드 권장**(자동 커버는 세로 디테일컷의 중앙이 잡혀 밋밋할 수 있음).

## 3. 업로드 & 연결 (Studio 순서)

1. **롱폼 먼저 업로드** (`long.mp4`)
   - 설명란에 `long.publish.md` 내용 붙여넣기 → **여기 링크는 클릭됨**. (공정위 문구 + `👉 구매:` 링크 포함)
   - **카드 추가:** 동영상 → 편집 → 카드 → 동영상/재생목록(또는 YPP면 연결된 사이트=랜딩).
   - **끝화면 추가:** 마지막 20초 구간에 "관련 영상/구매" + "구독" 요소 배치(세이프존이 비어 있어 본문과 안 겹침).
2. **쇼츠 업로드** (`video.mp4`)
   - Studio → 콘텐츠 → 해당 쇼츠 → **관련 동영상** = 방금 올린 롱폼 지정 → 저장.
   - (쇼츠 설명/댓글 링크는 어차피 비클릭이므로 "👇 더보기/관련영상에서 자세히" 문구만)
3. 결과: 쇼츠 시청자가 관련 동영상 탭 → 롱폼 → 설명란/카드/끝화면 클릭 → 랜딩 → 제휴몰.

### 롱폼 설명란 템플릿 (클릭됨)
```

<훅 한 줄>
구매 링크는 더보기란·카드를 확인하세요

👉 구매: https://naver.me/xxxx

#브랜드 #카테고리 #바로쇼핑
```

## 4. 요건 체크리스트

| 항목 | 요건 |
|---|---|
| 쇼츠 → 관련 동영상 연결 | 고급 기능 액세스(채널 인증) |
| 끝화면 표시 | 영상 ≥ 25초 (ProductLong 60~90초 → 충족) |
| 설명란 클릭 링크 | 요건 없음 (롱폼은 항상 클릭) |
| 카드/끝화면 → 외부 사이트 링크 | YPP(구독 1천·시청 4천h) + 연결된 사이트 인증 |
| 목적지가 쇼츠로 재분류 방지 | 가로 16:9 또는 세로 3분 초과 |

> YPP 전이라도 **롱폼 설명란 클릭 링크**만으로 퍼널은 성립한다. 카드/끝화면 외부링크는 YPP 달성 후 추가.
